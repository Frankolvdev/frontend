# 02B — Visual Motion Engine

ZIP incremental sobre 02A.

Incluye:

- sistema global de iluminación reactiva;
- textura cinematográfica;
- barra de progreso de scroll;
- dock flotante de estado;
- componente Reveal con IntersectionObserver;
- componente Magnetic;
- componente ParallaxLayer;
- nueva escena Experience Lab;
- selector interactivo de looks;
- radar animado;
- telemetría dinámica;
- simulación de generación.

No agrega dependencias externas. Todo funciona con React, Canvas/CSS y APIs del navegador.
Esto evita modificar package-lock.json antes de incorporar GSAP o React Three Fiber en
un módulo dedicado y verificable.
